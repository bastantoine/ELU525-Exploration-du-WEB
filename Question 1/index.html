Représentation des rôles du HTML, du CSS et du Javascript

Tous internet utilisent HTML et CSS, et la majorité Javascript aussi. Ces trois langages ont différents utilités :

  • HTML sert à décrire le contenu et la structure de la page
  • CSS sert à décrire à quoi va ressembler la page
  • Javascript permet de rendre la page dynamique

Représentation imagée des différentes parties du web

Le web est composé de plusieurs parties :

  • Le web surfacique qui comprend l'ensemble des pages indexées par les moteurs de recherche.
  • Le web profond (ou deep web) qui comprend l'ensemble des pages non indexées par les moteurs de recherche.
  • Le web obscur (ou dark web) qui comprend l'ensemble des pages existantes sur des réseaux parallèles à Internet, souvent pair-à-pair, dont le contenu est majoritairement illégal car l'utilisation est anonyme.
